# StockWidget 📈

Android home-screen widget for tracking stocks, indices, and cryptocurrencies in real time.

The app allows users to select which assets they want to monitor, and displays live prices and daily percentage changes directly on the Android home screen.

## Features

* Real-time market data using the Finnhub API
* Customizable Android widget
* User selection of stocks, indices, and crypto assets
* Daily percentage change indicators
* Clean dark-mode inspired UI
* Hebrew RTL support
* Persistent user preferences

## Technologies Used

* Kotlin
* Android Studio
* XML Layouts
* Android App Widgets
* Finnhub REST API
* SharedPreferences
* RecyclerView

## How It Works

The user selects desired assets from a predefined list inside the application.

The widget then fetches live market data from the Finnhub API and displays:

* Current price/index value
* Daily percentage change
* Positive/negative movement indicators

The widget updates dynamically and presents the data directly on the Android home screen.
